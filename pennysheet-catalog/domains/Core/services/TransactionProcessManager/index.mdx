## Overview

The Transaction Process Manager runs the transaction import flow, coordinating repeated calls to the [[service|EnableBankingAPIGateway]] until the import finishes.

- [[event|ImportTransactionsRequested]]: starts a new flow and sends [[command|FetchTransactions]] to the [[service|EnableBankingAPIGateway]].
- [[event|TransactionImportRetryRequested]]: restarts the flow for a previously failed import and sends [[command|FetchTransactions]] to the [[service|EnableBankingAPIGateway]].
- [[event|TransactionImportContinued]]: progresses the flow and sends [[command|FetchTransactions]] again, using the continuation key returned by the Enable Banking API.
- [[event|TransactionImportCompleted]] / [[event|TransactionImportFailed]]: end the current flow. No further [[command|FetchTransactions]] is sent.
