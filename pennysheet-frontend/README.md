# Pennysheet Frontend

A web client built with React.

## Project Structure

```
pennysheet-frontend/src/
├── api/          # Axios client to work with backend API.
├── components/   # React components.
├── hooks/        # Hooks mainly for data from projections.
└── pages/        # Pages for each route.
└── App.tsx
└── index.css
└── main.tsx
```
